# Assets de DOMANI (imágenes y video)

Sube aquí las imágenes y el video del diseño. **Los nombres deben ser
EXACTOS** (en minúsculas) para que el código los encuentre. Se sirven en
`https://domani.club/assets/<archivo>`.

## Archivos requeridos por la landing

| Archivo | Uso | Formato sugerido |
|---|---|---|
| `hero.png` | Fondo del hero (Valente en el salón, horizontal) | PNG/JPG, ~1920px ancho |
| `domani-intro.mp4` | Video de la sección 2 | MP4 H.264, 1920×1080 |
| `via-casino.png` | Tarjeta "Los Casinos" | PNG/JPG, vertical 3:4 |
| `via-destreza.png` | Tarjeta "Juegos de destreza" | PNG/JPG, vertical 3:4 |
| `via-academia.png` | Tarjeta "La Academia" | PNG/JPG, vertical 3:4 |
| `via-mercado.png` | Tarjeta "El Mercado" | PNG/JPG, vertical 3:4 |
| `promesa-bg.png` | Fondo sección "La Promesa" (valet + auto) | PNG/JPG, horizontal |
| `valente_cut.png` | Recorte del anfitrión Domenico Valente | PNG con transparencia |
| `entrada-b.png` | Fondo CTA final — **escritorio** (dos guardias) | PNG/JPG, 1672×941 |
| `entrada-a.png` | Fondo CTA final — **móvil** (un guardia) | PNG/JPG, 941×1672 |
| `aurelio.png` | Moneda Aurelio (círculo con transparencia) | PNG con transparencia |

> El isologo "D" y los escudos de las Casas (HouseCrest) NO son imágenes:
> se generan en CSS/SVG dentro del código. No hay que subirlos.

## Nota sobre el video
GitHub por la web admite archivos de hasta **25 MB**. Si `domani-intro.mp4`
pesa más, avísame y lo comprimimos o usamos otra vía (no conviene meter
videos muy pesados directo al repo).
